🎮 Otaku Quiz Game — README
📌 Overview
Otaku Quiz Game is a fast‑paced anime trivia speedrun game built with HTML, CSS, and JavaScript.
It includes:

⏱ Speedrun Mode Only (60‑second challenge)

👤 Player Name Input

🎞 Smooth Fade Animations

🧠 Multiple Difficulty Levels

🖼 Custom Anime Images per Question

💾 LocalStorage Save System

🏅 Always‑Visible Live Leaderboard

🔁 Restart System

📱 Responsive & Accessible UI
